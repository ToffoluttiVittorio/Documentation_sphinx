Introduction

Le module cartographique de cette plateforme permet de présenter ces couches de données dans un environnement plus technique. Cette interface permet de représenter plusieurs couches géographique mais ne peut pas se substituer à l’utilisation complète d’un outil SIG bureautique type QGIS.
